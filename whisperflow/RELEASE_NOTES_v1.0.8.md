# Verbal v1.0.8 Release Notes

## UI Improvements
- Completely redesigned listening widget with smoother, less boxy appearance
- Enhanced visual effects with improved shadows, glows, and 3D effects
- Larger and more responsive waveform visualization
- Better text readability with subtle shadows
- Smoother animations and transitions

## Bug Fixes
- Fixed application crashes caused by memory leaks in audio recording
- Improved WebSocket connection stability for cross-device sync
- Added proper error handling to prevent crashes from unhandled exceptions
- Implemented automatic reconnection for sync service
- Limited audio buffer size to prevent memory exhaustion
- Fixed update check frequency to prevent resource consumption

## Improvements
- Enhanced application stability and reliability
- Better error recovery for network connectivity issues
- Improved resource management to prevent crashes
- More robust background thread handling

## How to Update
This version will automatically update if you have auto-update enabled. If not, you can manually download the latest version from our releases page.
